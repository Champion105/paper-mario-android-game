# Paper Story RPG 📖✨

**Paper Story RPG** is a stylized, paper-aesthetic RPG referencing the classic style of *The Thousand-Year Door*. Explore a 3D world with 2D "paper" characters, engage in tactical turn-based battles, and experience a unique visual charm.

## 🚀 Getting Started

### Prerequisites
- [Android Studio](https://developer.android.com/studio) (latest version recommended)
- Android SDK 34+
- A physical Android device or Emulator

### How to Open the Project
1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Open **Android Studio**.
3. Select **File > Open**.
4. Navigate to the `Paper Story RPG` root folder and click **OK**.
5. Wait for Gradle to sync.
6. Press **Run** (Shift + F10) to start the game on your device/emulator.

## 🛠️ Main Roadmap Goals

We are currently in the prototype phase, focusing on authentic TTYD-style mechanics:

- [x] **Battle Menu Overhaul**: Implemented a mathematically precise radial arc menu with aspect-ratio-locked icons.
- [x] **Gamepad Support**: Robust detection for physical controllers (ROG Ally X, Xbox, etc.) with automatic UI toggling.
- [x] **Enemy HP UI**: Stylized numeric HP display with heart icons, anchored to the stage at the enemy's feet.
- [ ] **Action Commands**: Implement timing-based "Great!" and "Excellent!" triggers for attacks and defense.
- [ ] **Camera Transitions**: Refine stage-to-overworld camera movement and dynamic battle zooms.

## 🎮 Controls
- **Touch**: Virtual joystick on left, "A" button on right.
- **Keyboard**: `WASD` to move, `Space` to jump/interact.
- **Gamepad**: Left Stick to move, `South` button (A/Cross) to jump/interact.
